Media Queries

Complete the three tasks to add styling to the page at different breakpoints

Task 1: Mobile-First

  1. Remove the margin on the body
  2. Change the font-family
  3. Make the font-size larger
  4. Add spacing between the list items

Task 2: Break point at 768px

  1. Add margin around the edges of the page
  2. Make the images display in two columns
  3. Change the background color of the page to a light grey

Task 3: Breakpoint at 1024px

  1. Make all the images display side-by-side
  2. Change the background color of the page to purple
  3. Adjust the font-size to be 2x the browser default (use the rem unit for that)
